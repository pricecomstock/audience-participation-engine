# audience-participation-engine

> A tool for allowing real-time audience smartphone participation through the web, designed for use in comedy shows but applicable to anything.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
